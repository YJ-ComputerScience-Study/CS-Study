# 스레드를 만드는 방법을 나열해주세요.

Java에서 Thread를 생성하는 방법은 두 가지가 존재합니다.

1. Thread 클래스를 상속받아 `run()` 메서드를 오버라이딩하는 것
2. Runnable 인터페이스를 implements하여 `run()` 메서드를 정의하고, 이를 구현한 클래스를 Thread의 매개변수로 넣어 Thread를 생성하는 것
