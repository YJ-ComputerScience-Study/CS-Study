# final은 추상 클래스를 수정할 때 사용할 수 있나요?

`final`은 추상 클래스 수정 시 사용할 수 없습니다. `final`과 `abstract`는 상반된 의미를 가지고 있기 때문입니다. Java에서 `final` 키워드는 변수, 메소드, 또는 클래스에 대한 변경이나 수정을 금지할 때 사용합니다. 클래스 앞에 붙인다면 상속이 불가능한 클래스가 되고, 메소드 앞에 붙인다면 오버라이드할 수 없는 메소드가 됩니다. 변수 앞에 붙인다면 값을 변경할 수 없는 변수가 됩니다. 

`abstract` 클래스는 자식 클래스가 상속하는 방식으로 사용되는데, `final` 키워드를 선언하면 해당 클래스를 상속할 수 없게 되므로 서로 충돌하기에 `final`은 사용할 수 없습니다.
