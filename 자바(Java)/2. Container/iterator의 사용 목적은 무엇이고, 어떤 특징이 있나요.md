# iterator의 사용 목적은 무엇인가요? 어떤 특징이 있죠?

여러 데이터 집합체를 다루기 위해선 각각 데이터를 읽는 방법을 알아야 하기에 각 컬렉션에 접근이 어려운데, Iterator를 사용할 시에는 **모두 동일한 방식으로 접근이 가능**합니다.

특징으로는

1. 반복 중 컬렉션이 수정되면 Exception이 발생
2. 일회용으로, 한 번 사용 후에는 다시 사용할 수 없음
3. `hasNext()` 메서드로 다음 요소 확인, `next()` 메서드로 다음 요소 가져오기, `remove()` 메서드로 현재 요소 제거 가능

이 있습니다.
