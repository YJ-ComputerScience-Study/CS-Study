# CS-Study

## 📝 스터디 방식

1. 각자 책을 바탕으로 내용을 공부한다.

2. 책에 없는 내용이라면, 따로 공부해서 정리한다.

3. 나올 것 같은 질문과 모범 답변 준비해서 주제에 맞는 폴더에 md 파일로 올린다.

4. 스터디 시간에 서로 질문하고, 답변한다.
