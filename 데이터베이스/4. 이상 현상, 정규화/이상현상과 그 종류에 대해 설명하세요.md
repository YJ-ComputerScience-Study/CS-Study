# 이상현상과 그 종류에 대해 설명하세요.

이상 현상이란 릴레이션을 잘못 설계하여 데이터를 삽입/수정/삭제할 때 논리적 오류가 생기는 현상을 말합니다. 이상 현상에는 크게 세 가지 종류가 있습니다.

1. 삽입 이상 (Insertion Anomaly) : 자료를 삽입할 때 의도하지 않은 자료까지 삽입해야만 테이블에 자료 추가가 가능한 현상입니다.
2. 삭제 이상 (Deletion Anomaly) : 어떤 정보를 삭제하면, 의도하지 않은 다른 정보까지 삭제되어 원하지 않는 정보의 손실이 발생하는 현상입니다.
3. 갱신 이상 (Update Anomaly) : 중복된 데이터 중 일부만 수정되어 데이터 모순이 발생하는 현상입니다.
